# 🌟 MorningStar Website

> **Status:** Projeto Finalizado (Marco de Estudo - 2024) 🚩

### 📖 Sobre o Projeto
Este site foi desenvolvido em **2024** como um projeto colaborativo entre mim e o [@Bryan1Daniel](https://github.com/Bryan1Daniel) (e seu amigo, [@IsaacBerdaky](https://github.com/IsaacBerdaky)). 

O objetivo principal foi colocar em prática os fundamentos de desenvolvimento web que estávamos explorando na época, focando na criação de uma interface limpa e funcional. Decidi manter o código em seu estado original para servir como um **registro da minha evolução técnica**.

### 🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando o "trio fundamental" do Front-end:
* **HTML5**: Para a estruturação semântica do conteúdo.
* **CSS3**: Para estilização, layout e design visual.
* **JavaScript**: Para pequenas interações e lógica de interface.

### 🧠 O que este projeto representa
Embora seja um projeto de início de carreira, ele foi fundamental para consolidar conceitos como:
1.  **Trabalho em Equipe:** Coordenação de tarefas e divisão de responsabilidades no código.
2.  **Versionamento com Git:** Uso de commits e pushes para manter o projeto atualizado entre os desenvolvedores.
3.  **Resolução de Problemas:** Enfrentar os primeiros desafios de layout e responsividade.

### 🚀 Evolução
Hoje, com quase 18 anos e mais experiência, olho para este código e já consigo identificar diversas melhorias que faria (como uso de frameworks ou CSS moderno). No entanto, o **MorningStar** continua aqui como um troféu do início da minha jornada na tecnologia.
